Python yield

生成器，按照一定算法生成一个序列。

斐波那契数列


          def fib(n):
              ret = []
              a, b = 0, 1
              while b < n:
                  ret.append(b)
                  a, b = b, a + b
          return ret 

          def fib(n):
              a, b = 1, 1
              while a < n:
                  yield a
                  a, b = b, a + b
惰性求值

特性

    调用生成器函数时函数体不执行，第一次通过next()调用时开始执行到yield表达式，然后终止， 直到下一次调用next()方法时继续执行后续程序。
